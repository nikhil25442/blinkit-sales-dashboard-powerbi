# 📊 Blinkit Sales Dashboard - Power BI

An interactive Power BI dashboard analyzing Blinkit grocery sales performance, KPIs, and outlet insights.

---

## 📌 Project Overview
This project presents a detailed sales analysis of Blinkit (India's last-minute delivery app) using Power BI to generate actionable insights from grocery data.

---

## 🎯 Objectives
- Analyze total sales and item distribution  
- Track key performance indicators  
- Understand outlet performance  
- Identify product and category trends  

---

## 📊 Dashboard Features
- 💰 Total Sales: $1.20M  
- 📦 Total Items: 9K  
- ⭐ Avg Rating: 3.92  
- 📈 Avg Sales: 140.99  
- 🏪 Outlet size & location analysis  
- 🍔 Item type & fat content breakdown  
- 📅 Year-wise outlet establishment trends  

---

## 🛠 Tools & Technologies
- Power BI  
- Power Query  
- Excel  

---

## 📈 Key Insights
- Tier 3 outlets generate highest revenue  
- Certain item categories dominate sales  
- Outlet size significantly impacts performance  
- Sales trend varies across years
